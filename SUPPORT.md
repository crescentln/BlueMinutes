# Support

This project is maintained as an independent open-source public-interest
initiative. It currently has no service-level agreement, commercial support
commitment, or official institutional backing.

- Use GitHub Issues for reproducible bugs, documentation gaps, and scoped
  feature proposals.
- Use the Issue templates and include synthetic reproduction material only.
- Follow `SECURITY.md` for vulnerabilities; never disclose a vulnerability or
  private meeting content in a public Issue.
- Review `README.md`, `docs/`, and existing Issues before requesting help.

The application is not an official record system and must not be treated as an
authoritative source of diplomatic positions. Users remain responsible for
reviewing evidence, confirming derived content, applying organizational policy,
and maintaining appropriate backups.
